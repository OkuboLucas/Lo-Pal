🧮 Calculadora JS/HTMl
----------------------------------------------------------------------------------------------------------------
 ## Projeto - Calculadora em JavaScript e HTML 
Um projeto minimalista de calculadora desenvolvida para praticar
os fundamentos de lógica de programação com JavaScript.

. Calculadora feita para uso básico e contas simples.

. Adição (+) / Subtração (-) / Divisão (/) / Multiplicação ( x )

🛠️ Tecnologias

**HTML**: Estrutura da página

**JavaScript**: Lógica de cálculo.
